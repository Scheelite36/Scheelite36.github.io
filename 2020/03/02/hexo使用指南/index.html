hexo使用指南（持续更新）

发表于2020-03-02|更新于2021-06-08|杂七杂八

关于hexo文章中插入图片的问题

  1. 安装cnpm

  2. 使用cnpm 安装0.05版本的hexo-asset-image插件

    该插件作用是创建博文时自动生成同名文件夹，用于存放图片

  3. 修改_config.yml文件，将post_asset_folder选项改为true

  4. 在文章用引用图片使用相对路径

hexo g 遇到的报错

YAMLException: can not read a block mapping entry

解决办法：文章中 tags: 后面一定要有半角空格！！！注意冒号后边要有空格，“-”符号前后都要有空格！cover: 后面也要有一个空格！

Template render error: (unknown path)

文章中出现了这类字符，但是没有放入代码区域中，尽量避免使用他们

hexo子分类

mathjax

在hexo主题中_config.yml文件中设置mathjax开启

然后需要在博客目录下安装渲染模块

local_search

现在博客目录下安装hexo-generator-search模块

然后再hexo的_config.yml中添加

在主题设置_config.yml中开启

若本地搜索依旧无法正常使用建议重装主题

mermaid

在确保主题支持的情况下开启mermaid

mermaid书写格式：

pie title Key elements in Product X "Calcium" : 42.96 "Potassium" : 50.05 "Magnesium" : 10.01 "Iron" : 5

使用mermaid一定要注意别直接写在文章结尾！！！，否则会报未知错误！

hexo博客中图片无法加载的问题

在把文章的hexo迁移到我的macbook上之后发生一系列恼人的问题，最严重最折磨人的就是到hexo博客上的图片不显示了，我看hexo有两种文章链接本地图片的方法，之前我一直使用的相对路径导入图片，也是typora支持的形式，这样编辑就方便了。hexo还有一种模板语言导入图片方法。

这两种方法其实对应着不同的hexo图片加载插件，分别通过两个命令安装
